개요 및 역사

개요

이 강의는 HTML(HyperText Markup Language)의 기초부터 심화 개념까지 포괄적으로 다루며, 웹 페이지를 효과적으로 설계하고 작성할 수 있는 능력을 배양하는 것을 목표로 합니다. HTML은 웹 페이지의 구조를 정의하고 콘텐츠를 조직하는 데 사용되는 기본적인 마크업 언어입니다. 웹 개발의 기초를 이해하는 것은 다양한 웹 기술을 활용하는 데 필수적입니다.

이 강의의 목표는 HTML의 기본 구조와 문서 작성법을 이해하고, 텍스트, 링크, 이미지, 리스트, 표와 같은 HTML 태그와 요소를 효과적으로 활용하여 웹 페이지를 구축하는 능력을 배양하는 것입니다. 또한, 사용자 입력을 처리할 수 있는 폼을 작성하고, 시멘틱 HTML을 통해 접근성과 SEO를 고려한 웹 페이지를 설계하는 방법을 익힙니다. HTML5의 새로운 기능을 활용하여 멀티미디어 콘텐츠를 포함한 현대적인 웹 페이지를 제작하며, 실습을 통해 이론을 실제로 적용하고 실력을 검증하는 것을 목표로 합니다.

역사

1. HTML의 탄생 (1991) HTML은 1991년에 팀 버너스리(Tim Berners-Lee)에 의해 처음 개발되었습니다. 그는 CERN에서 연구 결과를 공유하기 위해 HTML을 사용하여 웹 문서를 작성하는 방법을 제안했습니다. 초기 HTML은 매우 단순하며, 기본적인 문서 구조와 하이퍼링크 기능만을 지원했습니다.
2. HTML 2.0 (1995) 1995년에 HTML 2.0이 발표되면서 웹의 표준화가 이루어졌습니다. HTML 2.0은 폼과 테이블 같은 새로운 기능을 추가하였고, 웹 페이지의 구조화와 상호작용을 가능하게 했습니다.
3. HTML 3.2 (1997) HTML 3.2는 마이크로소프트와 넷스케이프의 협력으로 표준화되었으며, 스타일 시트와 자바스크립트 지원을 포함하여 웹 페이지의 디자인과 동적 기능을 개선했습니다.
4. HTML 4.0 (1999) HTML 4.0은 시멘틱 구조와 접근성을 개선하기 위해 중요한 업그레이드가 이루어졌습니다. 이 버전에서는 div, span, header, footer와 같은 시멘틱 태그가 도입되었으며, CSS를 통해 스타일링을 별도로 처리하도록 권장했습니다.
5. HTML5 (2014) HTML5는 웹 표준의 큰 전환점을 의미합니다. 2014년에 공식적으로 발표된 HTML5는 비디오, 오디오, 캔버스, 웹 스토리지 등 새로운 기능을 추가하여 보다 풍부하고 다채로운 웹 애플리케이션을 가능하게 했습니다. HTML5는 또한 웹 접근성과 성능 향상을 목표로 하여 모바일 장치와 다양한 플랫폼에서 일관된 사용자 경험을 제공하도록 설계되었습니다.
6. 현재와 미래 현재 HTML은 지속적으로 발전하고 있으며, 최신 웹 표준을 지원하고 있습니다. 웹 개발자들은 최신 HTML 표준을 활용하여 더욱 매력적이고 기능적인 웹 페이지를 만들고 있습니다. HTML의 발전은 웹 기술의 진화와 함께 계속될 것입니다.
